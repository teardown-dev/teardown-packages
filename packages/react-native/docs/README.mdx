# Teardown React Native SDK Documentation

Complete documentation for integrating and using the Teardown React Native SDK in your mobile applications.

## Overview

Teardown SDK provides a complete solution for managing:
- **User Identity** - Anonymous and authenticated user tracking
- **Session Management** - Persistent sessions across app launches
- **Force Updates** - Automatic version checking and enforcement
- **Device Information** - Hardware, OS, and app metadata
- **Analytics Ready** - Built-in event tracking foundation

## Quick Links

### Installation

```bash
bun add @teardown/react-native
```

### Basic Setup

```typescript
import { TeardownCore, TeardownProvider } from "@teardown/react-native";

const teardown = new TeardownCore({
  api: {
    org_id: "your-org-id",
    project_id: "your-project-id",
    api_key: "your-api-key",
    environment_slug: "production",
  },
  storage: { createStorage },
  device: { adapter: new ExpoDeviceAdapter() },
});

function App() {
  return (
    <TeardownProvider core={teardown}>
      {/* Your app */}
    </TeardownProvider>
  );
}
```

### Using in Components

```typescript
import { useTeardown, useForceUpdate } from "@teardown/react-native";

function MyComponent() {
  const { core } = useTeardown();
  const { isUpdateRequired } = useForceUpdate();

  // Use SDK features
}
```

## Key Features

### 🔐 Identity Management
- Anonymous device tracking
- User persona association
- Persistent sessions
- Event-driven state updates

### 📱 Device Information
- Platform detection (iOS/Android)
- Hardware details
- OS information
- App version tracking
- OTA update metadata

### 🔄 Force Updates
- Automatic version checking
- Configurable check frequency
- App foreground detection
- Blocking and non-blocking updates
- React hook integration

### 💾 Storage
- Namespaced persistence
- MMKV support
- AsyncStorage support
- Custom storage backends
- Type-safe wrappers

### 📊 Logging
- Structured logging
- Debug mode
- Named loggers
- Multiple log levels

## Platform Support

- ✅ React Native (iOS/Android)
- ✅ Expo (Managed & Bare workflow)
- ✅ Expo Go (with limitations)

## Requirements

- React Native >= 0.64
- React >= 16.8 (hooks support)
- TypeScript >= 4.5 (recommended)
