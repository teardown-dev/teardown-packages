# API Reference

Complete reference for all Teardown SDK classes and methods.

## TeardownCore

The main SDK orchestrator.

### Constructor

```typescript
new TeardownCore(options: TeardownCoreOptions)
```

**Options:**

```typescript
type TeardownCoreOptions = {
  logging?: LoggingClientOptions;
  api: ApiClientOptions;
  storage: StorageClientOptions;
  identity: IdentityClientOptions;
  device: DeviceClientOptions;
  forceUpdate?: ForceUpdateClientOptions;
};
```

### Properties

```typescript
core.api: ApiClient              // API client instance
core.device: DeviceClient        // Device client instance
core.identity: IdentityClient    // Identity client instance
core.forceUpdate: ForceUpdateClient  // Force update client instance
```

### Methods

#### shutdown()

Clean up resources and close connections.

```typescript
shutdown(): void
```

**Example:**
```tsx
core.shutdown();
```

## IdentityClient

Manages user and device sessions.

### Constructor

```typescript
new IdentityClient(
  logging: LoggingClient,
  utils: UtilsClient,
  storage: StorageClient,
  api: ApiClient,
  device: DeviceClient,
  options: IdentityClientOptions
)
```

**Options:**

```typescript
type IdentityClientOptions = {
  identifyOnLoad?: boolean; // Default: false
};
```

### Methods

#### identify()

Identify a user or device.

```typescript
identify(persona?: Persona): AsyncResult<IdentityUser>
```

**Parameters:**

```typescript
type Persona = {
  name?: string;
  user_id?: string;
  email?: string;
};
```

**Returns:**

```typescript
type AsyncResult<IdentityUser> = 
  | { success: true; data: IdentityUser }
  | { success: false; error: string }

type IdentityUser = {
  session_id: string;
  device_id: string;
  persona_id: string;
  token: string;
  version_status?: VersionStatusEnum;
};
```

**Example:**
```tsx
const result = await core.identity.identify({
  user_id: '123',
  email: 'user@example.com',
});

if (result.success) {
  console.log('Session:', result.data.session_id);
}
```

#### refresh()

Refresh current session (requires active session).

```typescript
refresh(): AsyncResult<IdentityUser>
```

**Example:**
```tsx
const result = await core.identity.refresh();
if (result.success) {
  console.log('Session refreshed');
}
```

#### reset()

Clear current session and return to unidentified state.

```typescript
reset(): void
```

**Example:**
```tsx
core.identity.reset();
```

#### getSessionState()

Get current session state.

```typescript
getSessionState(): SessionState
```

**Returns:**

```typescript
type SessionState = 
  | { type: 'unidentified' }
  | { type: 'identifying' }
  | { type: 'identified'; session: Session; version_status: VersionStatusEnum }
```

**Example:**
```tsx
const state = core.identity.getSessionState();
if (state.type === 'identified') {
  console.log(state.session);
}
```

#### onSessionStateChange()

Subscribe to session state changes.

```typescript
onSessionStateChange(listener: (state: SessionState) => void): () => void
```

**Returns:** Unsubscribe function

**Example:**
```tsx
const unsubscribe = core.identity.onSessionStateChange((state) => {
  console.log('Session changed:', state.type);
});

// Later: cleanup
unsubscribe();
```

#### shutdown()

Remove all event listeners.

```typescript
shutdown(): void
```

## ForceUpdateClient

Manages version checking and force updates.

### Constructor

```typescript
new ForceUpdateClient(
  logging: LoggingClient,
  storage: StorageClient,
  identity: IdentityClient,
  options?: ForceUpdateClientOptions
)
```

**Options:**

```typescript
type ForceUpdateClientOptions = {
  throttleMs?: number;              // Default: 30000
  checkCooldownMs?: number;         // Default: 300000
  identifyAnonymousDevice?: boolean; // Default: false
};
```

### Methods

#### getVersionStatus()

Get current version status.

```typescript
getVersionStatus(): VersionStatus
```

**Returns:**

```typescript
type VersionStatus =
  | { type: 'unknown' }
  | { type: 'checking' }
  | { type: 'up_to_date' }
  | { type: 'update_available' }
  | { type: 'update_required' }
```

**Example:**
```tsx
const status = core.forceUpdate.getVersionStatus();
console.log(status.type);
```

#### onVersionStatusChange()

Subscribe to version status changes.

```typescript
onVersionStatusChange(listener: (status: VersionStatus) => void): () => void
```

**Returns:** Unsubscribe function

**Example:**
```tsx
const unsubscribe = core.forceUpdate.onVersionStatusChange((status) => {
  if (status.type === 'update_required') {
    showUpdateModal();
  }
});
```

#### shutdown()

Remove all event listeners and cleanup.

```typescript
shutdown(): void
```

## StorageClient

Manages persistent local storage.

### Constructor

```typescript
new StorageClient(
  logging: LoggingClient,
  options: StorageClientOptions
)
```

**Options:**

```typescript
type StorageClientOptions = {
  adapter: StorageAdapter;
};

type StorageAdapter = {
  getItem: (key: string) => Promise<string | null>;
  setItem: (key: string, value: string) => Promise<void>;
  removeItem: (key: string) => Promise<void>;
};
```

### Methods

#### createStorage()

Create namespaced storage.

```typescript
createStorage(namespace: string): SupportedStorage
```

**Returns:**

```typescript
interface SupportedStorage {
  getItem(key: string): string | null;
  setItem(key: string, value: string): void;
  removeItem(key: string): void;
}
```

**Example:**
```tsx
const storage = core.storage.createStorage('myFeature');
storage.setItem('key', 'value');
const value = storage.getItem('key');
```

#### shutdown()

Cleanup storage connections.

```typescript
shutdown(): void
```

## Logger

Individual logger instance.

### Methods

#### info()

Log informational message.

```typescript
info(message: string, ...args: unknown[]): void
```

**Example:**
```tsx
logger.info('User logged in', { userId: '123' });
```

#### error()

Log error message.

```typescript
error(message: string, ...args: unknown[]): void
```

**Example:**
```tsx
logger.error('Request failed', error);
```

#### debug()

Log debug message (only if debug enabled).

```typescript
debug(message: string, ...args: unknown[]): void
```

**Example:**
```tsx
logger.debug('Processing item', item);
```

#### warn()

Log warning message.

```typescript
warn(message: string, ...args: unknown[]): void
```

**Example:**
```tsx
logger.warn('Deprecated API', { method: 'oldMethod' });
```

#### trace()

Log stack trace.

```typescript
trace(message: string, ...args: unknown[]): void
```

**Example:**
```tsx
logger.trace('Execution path');
```

#### setDebugEnabled()

Enable or disable debug logging.

```typescript
setDebugEnabled(enabled: boolean): void
```

**Example:**
```tsx
logger.setDebugEnabled(true);
```

#### isDebugEnabled()

Check if debug logging is enabled.

```typescript
isDebugEnabled(): boolean
```

**Example:**
```tsx
if (logger.isDebugEnabled()) {
  logger.debug('Debug info');
}
```

## Type Definitions

### DeviceInfo

```typescript
type DeviceInfo = {
  application: {
    version: string;
    build_number: number;
  };
  hardware: {
    device_name: string;
    device_brand: string;
    device_type: string;
  };
  os: {
    platform: DevicePlatformEnum;
    name: string;
    version: string;
  };
  update: {
    is_enabled: boolean;
    update_id: string;
    update_channel: string;
    runtime_version: string;
    emergency_launch: {
      is_emergency_launch: boolean;
      reason?: string;
    };
    is_embedded_launch: boolean;
    created_at: string;
  } | null;
  notifications: {
    push: {
      platform: NotificationPlatformEnum;
      enabled: boolean;
      granted: boolean;
      token?: string;
    };
  };
};
```

### Enums

```typescript
enum DevicePlatformEnum {
  IOS = "IOS",
  ANDROID = "ANDROID",
  WEB = "WEB",
}

enum NotificationPlatformEnum {
  APNS = "APNS",
  FCM = "FCM",
  EXPO = "EXPO",
}

enum IdentifyVersionStatusEnum {
  UP_TO_DATE = "UP_TO_DATE",
  UPDATE_AVAILABLE = "UPDATE_AVAILABLE",
  UPDATE_REQUIRED = "UPDATE_REQUIRED",
}
```

### AsyncResult

```typescript
type AsyncResult<T> = 
  | { success: true; data: T }
  | { success: false; error: string }
```
